# 1009.Assertion

**签到题**

## 题目大意
多组数据
给定m件物品，分成n组
问：是否无论怎么分都至少有一组个数超过d

## 解题思路
平均分组，找最大那个组的个数和d比较

## 参考程序
> 参考代码为已AC代码主干，其中部分功能需读者自行实现

```cpp
int solve()
{
    ll m,n,d;
    cin >> n >> m >> d;
    if((m-1)/n+1>=d) cout << YES;
    else cout << NO;
    return 0;
}
```
